# Docs Overview

Bu klasor proje icin hala faydali olan teknik ve operasyonel dokumanlari tutar.

Korunan ana dokumanlar:

- `architecture/`: teknik mimari notlari
- `migrations/`: migration ve veri tasima notlari
- `plans/`: aktif teknik planlar
- `chat-api.md`: chat modulu referansi
- `backlog.md`: sonraki iyilestirmeler

Bu depoda onceki boilerplate/planning surecinden kalan detayli PRD, epic ve story dokumanlari temizlenmistir.
